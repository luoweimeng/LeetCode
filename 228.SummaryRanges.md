## Problem setting

Given a sorted integer array without duplicates, return the summary of its ranges.

```
Example 1:
Input: [0,1,2,4,5,7]
Output: ["0->2","4->5","7"]
Example 2:
Input: [0,2,3,4,6,8,9]
Output: ["0","2->4","6","8->9"]
```

## Approach 1

```java
class Solution {
    public List<String> summaryRanges(int[] nums) {
        int i = 0;
        int j = 0;
        List<String> result = new ArrayList<>();
        while (j < nums.length) {
            if (j == 0 || nums[j] == nums[j-1] + 1)
                j++;
            else {
                if (j == i + 1)
                    result.add(String.valueOf(nums[i]));
                else
                    result.add(String.valueOf(nums[i]) + "->" + String.valueOf(nums[j-1]));
                i = j;
                j++;
            }
        }
        
        //add trailing characters
        if (i < j) {
            if (j == i + 1)
                result.add(String.valueOf(nums[i]));
            else
                result.add(String.valueOf(nums[i]) + "->" + String.valueOf(nums[j-1]));
        }
        return result;
    }
}
```

## Approach 2

```java
class Solution {
    public List<String> summaryRanges(int[] nums) {
        int i = 0;
        int j = 0;
        int n = nums.length;
        List<String> result = new ArrayList<>();
        while (i < n) {
            j = i + 1;
            while (j < n && nums[j] == nums[j-1] + 1)
                j++;
            if (j == i + 1)
                result.add(String.valueOf(nums[i]));
            else
                result.add(String.valueOf(nums[i]) + "->" + String.valueOf(nums[j-1]));
            i = j;
        }
        
        return result;
    }
}
```
